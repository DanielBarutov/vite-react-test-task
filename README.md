## 🇬🇧 English Description

### 🌟 Overview

This project is a high-performance product creation dashboard developed as a technical assessment. It features intelligent automation using AI to streamline the e-commerce workflow.

**The core functionality:** The user enters a product name, and the system automatically generates:

- **Categories:** Smart classification.
- **Description:** Marketing-ready copy.
- **SEO Data:** Optimized titles and keywords for search engines.

### 🛠 Tech Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) (Lightning fast build)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/) (Radix UI based)
- **AI Integration:** Google Gemini AI (Generative Language API)
- **State Management:** React Hooks / Context API
- **Deployment:** [Vercel](https://vercel.com/)

### 🚀 Key Features

- **AI-Powered Content:** Instant generation of SEO and metadata from a single product name.
- **Modern UX:** Clean, responsive interface built with shadcn/ui.
- **Validation:** Robust form handling and real-time feedback.
- **Performance:** Optimized assets and minimal bundle size thanks to Vite.

## 🇷🇺 Описание проекта

### 🌟 Обзор

Данный проект представляет собой высокопроизводительную панель управления созданием товаров. Это кейс, разработанный в рамках тестового задания, демонстрирующий интеграцию нейросетей в стандартные бизнес-процессы.

**Основная фишка:** Пользователь вводит только название товара, а система автоматически генерирует:

- **Категории:** Умная классификация товара.
- **Описание:** Продающий текст, созданный AI.
- **SEO-данные:** Оптимизированные заголовки и ключевые слова.

### 🛠 Стек технологий

- **Фреймворк:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Стилизация:** [Tailwind CSS](https://tailwindcss.com/)
- **Компоненты:** [shadcn/ui](https://ui.shadcn.com/)
- **AI Интеграция:** Google Gemini AI
- **Деплой:** [Vercel](https://vercel.com/)

### 🚀 Ключевые возможности

- **AI Генерация:** Мгновенное создание контента и метаданных на основе одного лишь названия.
- **Современный UX:** Чистый и отзывчивый интерфейс на базе библиотеки shadcn.
- **Валидация:** Обработка форм и мгновенная обратная связь.
- **Скорость:** Быстрая сборка и работа приложения благодаря Vite.

### ⚙️ Установка и запуск (Local Setup)

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/DanielBarutov/vite-react-test-task.git
   ```

2. Установить зависимости:

   ```bash
   npm install
   ```

3. Создать файл .env и добавить ваш ключ:
   ```bash
   VITE_GOOGLE_AI_API_KEY=your_key_here
   ```
4. Запустить проект:
   ```bash
   npm run dev
   ```
